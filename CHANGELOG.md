# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-09

### Added
- Initial release of active_dataflow-connector-source-json_rpc
- JsonRpcSource class for receiving data via JSON-RPC server
- Automatic server lifecycle management (start/stop)
- Thread-safe queue-based record buffering
- Batch processing support
- Custom handler class support
- Server health monitoring and queue size tracking
- JSON serialization/deserialization support
- RSpec test suite with comprehensive coverage
- Cucumber feature tests for behavior verification
- Complete documentation and usage examples
